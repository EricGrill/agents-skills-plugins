# Contributing to Agents Skills Plugins

## Quick Start

```bash
# Clone and setup
git clone https://github.com/EricGrill/agents-skills-plugins.git
cd agents-skills-plugins

# Install Claude Code and follow setup instructions
```

## Adding a New Skill

1. Create a new directory under `skills/`
2. Add SKILL.md with usage instructions
3. Include any scripts or assets
4. Update the main README

## Pull Request Process

1. Ensure your skill follows the naming convention
2. Add documentation
3. Test with Claude Code
4. Submit PR with description
